# 🎵 Spotify Backend

A scalable backend for a Spotify-inspired music streaming platform built using Node.js, Express, MongoDB, JWT Authentication, and modern backend engineering practices.

---

# 🚀 Project Vision

This project aims to replicate and expand core backend functionalities of Spotify including:

* Authentication & Authorization
* Music Streaming APIs
* Artist Upload System
* Albums & Playlists
* Likes & Recently Played
* Search & Recommendation System
* Admin Controls
* Real-time Features
* Scalable Architecture

The goal is to build a production-style backend suitable for learning real-world backend engineering and system design concepts.

---

# 🛠️ Tech Stack

## Backend

* Node.js
* Express.js

## Database

* MongoDB

* JWT (JSON Web Tokens)

# 🔧 Installation

```bash
git clone https://github.com/Rajtiwari0202/Spotify_Backend.git

cd Spotify_Backend

npm install
```

---

# ⚙️ Environment Variables

Create a `.env` file:

```env
PORT=3000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

---

# ▶️ Run Project

```bash
npm run dev
```
